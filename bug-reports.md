## Bug 1 — High Priority
**Summary:** Login fails with valid credentials
**Site:** saucedemo.com
**Priority:** High
**Status:** Done ✅

**Steps to Reproduce:**
1. Go to saucedemo.com
2. Enter username: standard_user
3. Enter password: secret_sauce
4. Click Login

**Expected Result:** User lands on home page
**Actual Result:** Error message is shown

## Bug 2 — Medium Priority
**Summary:** No error message shown for empty password field
**Site:** saucedemo.com
**Priority:** Medium
**Status:** Done ✅

**Steps to Reproduce:**
1. Go to saucedemo.com
2. Leave username and password empty
3. Click Login

**Expected Result:** Error message "Password is required" appears
**Actual Result:** No error message is shown

---

## Bug 3 — Low Priority
**Summary:** Cart count does not update after removing item
**Site:** saucedemo.com
**Priority:** Low
**Status:** Done ✅

**Steps to Reproduce:**
1. Go to saucedemo.com
2. Login with username: standard_user
3. Password: secret_sauce
4. Click on any product
5. Click "Add to Cart"
6. Go to cart
7. Click "Remove" on the item

**Expected Result:** Cart count goes back to 0
**Actual Result:** Cart count does not update
